# 향상된 FOR문

```java
public class.UpgradeFor { 
  public static void main (String[] args) {
     int[] scores = {88,90,92,78,89};
     
     int sum = 0;
     for(int score:scores) {
     sum= sum + score;
     }
     
      system.out.println("점수 합계= " + sum);
     
     
      double avg= (double) sum / scores.length ;
      system.out.println("점수 평균= " + avg);
  }

}

```
